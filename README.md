Banking Fraud Monitoring System

This project simulates a simple banking fraud monitoring system.

The database stores information about customers, their accounts, cards, and transactions. Each transaction has a status and a risk score. Transactions marked as suspicious can generate fraud alerts for further investigation.

The system includes functions for customer and transaction analysis, procedures for account and transaction management, a trigger that automatically creates fraud alerts, and views for easier data analysis. A materialized view is used to provide a summary dashboard of transaction statistics and fraud-related information.
